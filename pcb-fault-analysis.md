# PCB Fault Analysis

## Objective
Identify faults in printed circuit boards (PCB) through inspection and electrical measurements.

## Inspection Process

1. Visual inspection of PCB.
2. Identification of damaged components.
3. Verification of solder joints.
4. Measurement of voltage levels.
5. Signal verification using oscilloscope.

## Common Failures

- Short circuits
- Broken PCB traces
- Cold solder joints
- Burnt components
- Voltage regulation failures

## Tools Used

- Digital multimeter
- Oscilloscope
- Rework station
- Magnification tools
